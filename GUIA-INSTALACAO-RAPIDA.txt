## 💻 OPÇÃO 2: USAR LOCALMENTE NO WINDOWS

2. Clique em "Download for Windows"

1. Clique em "Iniciar"


## 🖥️ OPÇÃO 3: USAR SERVIDOR LOCAL (RECOMENDADO PARA DESENVOLVIMENTO)

### COM PYTHON 3 (Mais Comum)

**Windows/Mac/Linux:**
```bash
# Entre na pasta do projeto
cd pixelarteofc2027.github.io

# Inicie o servidor
python -m http.server 8000
```

**Acesse:**
```
http://localhost:8000
```

---

### COM PYTHON 2 (Versão Antiga)

```bash
cd pixelarteofc2027.github.io
python -m SimpleHTTPServer 8000
```

**Acesse:**
```
http://localhost:8000
```

---

### COM NODE.JS (http-server)

**Passo 1: Instale Node.js**
- https://nodejs.org/

**Passo 2: Instale http-server**
```bash
npm install -g http-server
```

**Passo 3: Inicie o servidor**
```bash
cd pixelarteofc2027.github.io
http-server
```

**Acesse:**
```
http://localhost:8080
```

---

### COM LIVE SERVER (VS Code)

**Passo 1: Abra VS Code**
- Download: https://code.visualstudio.com/

**Passo 2: Instale Live Server**
- Vá em Extensions (Ctrl + Shift + X)
- Procure por "Live Server"
- Clique em "Install"

**Passo 3: Use Live Server**
- Abra index.html
- Clique direito
- Selecione "Open with Live Server"

**Pronto!** Abrirá em `http://localhost:5500/`

## 📞 PROBLEMAS COMUNS - SOLUÇÕES RÁPIDAS

### ❌ "Arquivo não encontrado"
```
Solução: Certifique-se que os 4 arquivos estão na mesma pasta
```

### ❌ "Porta 8000 já em uso"
```bash
# Use outra porta
python -m http.server 8080
```

### ❌ "Python não reconhecido"
```
Solução: Reinstale Python
https://www.python.org/downloads/
```

### ❌ "Git não reconhecido"
```
Solução: Reinstale Git
https://git-scm.com/download/
```

### ❌ "Imagens não carregam"
```
Solução: Use URLs completas ou imagens na mesma pasta
```

### ❌ "Animações não funcionam"
```
Solução: Verifique console (F12) para erros
```

### Opção 1: GitHub Pages (GRÁTIS - Recomendado)

**Passo 1:** Crie conta em https://github.com

**Passo 2:** Crie novo repositório
- Nome: `seu-usuario.github.io`
- Tipo: Public
- Clique "Create repository"

**Passo 3:** Envie seus arquivos
```bash
git init
git add .
git commit -m "Meu portfólio"
git branch -M main
git remote add origin https://github.com/seu-usuario/seu-usuario.github.io.git
git push -u origin main
```

**Passo 4:** Acesse
```
https://seu-usuario.github.io/
```
